# Professional Portfolio

A stunning, modern portfolio website with advanced animations and effects.

## Features

- 🎨 Modern dark theme design
- ✨ Smooth scroll animations
- 🎭 Custom cursor effects
- 📱 Fully responsive
- ⚡ Lightning fast with Vite
- 🎯 Interactive project cards
- 💫 Loading screen animation

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide Icons

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
